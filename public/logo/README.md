# Altair Medical System - Logo Assets

This directory contains the Altair Medical System logo files.

## Required Files

Please add the following logo files to this directory:

- `logo.svg` - Main logo in SVG format (preferred)
- `logo.png` - Main logo in PNG format (transparent background)
- `logo-light.svg` - Light variant for dark backgrounds
- `logo-dark.svg` - Dark variant for light backgrounds
- `logo-monochrome.svg` - Monochrome version
- `favicon.ico` - Favicon (16x16, 32x32, 48x48)
- `favicon.svg` - SVG favicon
- `apple-touch-icon.png` - Apple touch icon (180x180)

## Logo Specifications

- **Primary Colors**: 
  - Dark Blue: `#163852` (Navy Steel)
  - Golden Bronze: `#B8860B`
  - Gold: `#D4AF37`
- **Tagline**: "Design to Perform. Build to Last..."

## Usage

Logo files should be referenced from `/logo/` path in the application.

Example:
```tsx
<Image src="/logo/logo.svg" alt="Altair Medical System" />
```
